## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Ruby: v3.0.0
- Rails: v6.1.3
- Postgres: v13.2
- Node v10.19.0 - v14.15.5
- Bundle v2.2.3
- Yarn

### Setup

Instal gems with:

```
bundle install
```

Run Yarn for the icons dependancies:

```
yarn install
```

Setup database and seeds with:

```
   rails db:setup
```

### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    rpsec
```
